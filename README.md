# pretty-girl
###一个收集妹纸图片的App，并且用了比较多的Rx库写
###开发目的
为了学习DataBinding, Dagger2, RxJava, RxBinding, Retrofit 2.0等知识,参考了[mr-mantou-android](https://github.com/oxoooo/mr-mantou-android)。
